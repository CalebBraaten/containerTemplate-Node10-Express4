# Express Node App
## What is this?
This is a template repo to make building cloud services easier. You can create your own repo based off of this one, write your application, and then deploy it to Google Cloud Run with the click of a button!
### Great! What do I do next?
 1. Make a copy of this template - [learn how here](https://help.github.com/en/github/creating-cloning-and-archiving-repositories/creating-a-repository-from-a-template)
 2. This repo contains some simple code for an [express.js app](http://expressjs.com/). Write your code and make your commit. 
 3. Click the magic button below to package your code into a container and get it setup on Google Cloud Run automagically.
[![Run on Google Cloud](https://deploy.cloud.run/button.svg)
](https://deploy.cloud.run)
